PERRY ANDERSON: IT CONSULTANT PORTFOLIO
X
WebFlow is by far one of the best HTML / CSS editors you will find on the Internet. There is no need to directly edit CSS files or fiddle with HTML layout issues any longer with this polished feature rich editor.
For a small monthly fee you can put together a professional website in an afternoon. This tool is everything Adobe Dreamweaver was suppose to be all in one neat little online package!
Parse is the Persistence engine that we've been waiting for since the inception of object-oriented programs trying to map to relational tables. For simple applications it meets all our database needs.
It's free to use for basic applications but a small monthly fee will apply for large commercial usage. An open source solution is available for porting to independent servers!
Apple is the maker of the iPhone and we do iPhone apps to match the website that you'd need for your business. iPhone apps can be written in Objective-C and Swift.
iPhone technology improves significantly each year. We specialize in writing iPhone apps to work seamlessly with both your website and Android platforms!
Android is a Linux platform for cell phones rivaling Apple's iPhone. Android apps are written in Java and can go toe to toe with anything that iPhone can produce.
Android technology improves steadily each year and can be found on popular cell phone makers such as HTC, Google Nexus, Sony Xperia, Samsung Galaxy among many others!
RedHat has been at the forefront of commercial Linux server development since the early days of Linux into the market place. It's servers are considered to be the most secure in the industry.
RedHat would be the best choice for production level servers both in terms of scalability and in customer service. But it can be expensive for start up companies.
Ubuntu has emerged as the most stable and feature rich Linux platform in the industry that is also free of royalty costs.
Ubuntu would be the best choice to launch a start up Internet company given the quality of software and ease of use!
JBoss is an open source backend database platform for simplifying complex database applications. It is by far the most popular J2EE platform with which to create and manage large B2B applications.
RedHat specializes in adding value to the JBoss platform. JBoss itself is free but a paid RedHat support package provides you with a professional staff ready to ensure your company's uptime!
RackSpace is a feature rich online server farm that specializes in commercial online servers that can be customized to the developer's most specific needs.
RackSpace is not free! However it is scalable and has all the features necessary for maintaining a server farm online complete with live 24/7/365 support!
